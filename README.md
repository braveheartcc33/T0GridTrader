# T0 Grid Trader - A股网格交易系统

> 基于 ATR 动态间距的 A 股 T+0 网格交易策略，支持任意股票/ETF配置

## 功能特点

- **动态网格间距**：根据时段自动调整（开盘宽松、尾盘收紧）
- **ATR 噪声过滤**：减少假信号
- **破网停机机制**：价格偏离过大时停止交易
- **尾盘强制平仓**：14:30 尾盘统一平仓
- **飞书推送**：实时交易信号推送

## 配置说明

编辑 `config.py` 即可配置任意股票或ETF：

```python
STOCK_CODE = "000825.SZ"    # 任意A股代码
STOCK_NAME = "太钢不锈"       # 股票名称
```

支持的标的：
- A股股票（沪深两市）
- ETF基金（如 510300.SH 沪深300ETF）

## 核心参数

| 参数 | 说明 |
|------|------|
| `GRID_COUNT` | 网格档位数（默认10档） |
| `SHARES_PER_GRID` | 每格买卖股数 |
| `ATR_PERIOD` | ATR计算周期 |
| `GRID_SPACING_RULES` | 分时段网格间距倍数 |

详细设计见 `docs/T0_DESIGN.md`
